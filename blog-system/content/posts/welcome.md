# 欢迎使用 Vue + Markdown 博客系统

这是一个基于 Vue 3 和 Markdown 的现代化博客系统，具有以下特点：

## 主要特性

- **Markdown 渲染**: 支持完整的 Markdown 语法，包括代码高亮、表格、引用等
- **主题切换**: 支持浅色、深色和护眼三种主题
- **响应式设计**: 完美适配桌面、平板和手机设备
- **目录导航**: 自动生成文章目录，支持点击跳转
- **阅读进度**: 实时显示阅读进度条
- **标签系统**: 支持文章标签分类和筛选
- **搜索功能**: 支持文章标题和内容搜索

## 开始使用

1. 在 `content/posts/` 目录下创建新的 Markdown 文件
2. 在文件顶部添加 Frontmatter 元数据
3. 使用 Markdown 语法编写内容
4. 启动开发服务器查看效果

## 技术栈

- Vue 3 + Composition API
- Vue Router 4
- Pinia
- Markdown-it
- Vite
