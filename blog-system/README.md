# Vue + Markdown 博客系统

一个功能强大、完全可移植的 Vue 3 + Markdown 博客系统。

## 特性

- 📝 完整的 Markdown 渲染支持
- 🎨 三种主题切换（浅色、深色、护眼）
- 📱 响应式设计
- 📑 自动生成目录导航
- 📊 阅读进度条
- 🏷️ 标签分类和筛选
- 🔍 文章搜索功能
- 🚀 快速构建和部署

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 文档

详细的移植和配置指南请查看 [移植教程](./MIGRATION.md)。

- Vue 3 + Composition API
- Vue Router 4
