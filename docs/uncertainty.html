📝 公式说明

A类不确定度

uA = √[Σ(xi - x̄)² / n(n-1)]

其中 x̄ 为测量平均值，n 为测量次数

B类不确定度

uB = Δ仪 / k

均匀分布 k=√3，正态分布 k=3，三角分布 k=√6

合成不确定度

uc = √(uA² + uB1² + uB2² + ...)

考虑相关性时：uc = √(u₁² + u₂² + 2ρu₁u₂)

扩展不确定度

U = k × uc

k=2 对应约95%置信水平
