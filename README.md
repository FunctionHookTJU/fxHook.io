# 宇佐见函钩 - 个人学习网站

## 👋 项目介绍
这是一个基于纯HTML、CSS和JavaScript开发的个人学习网站，专注于算法可视化、编程学习笔记和技术分享。网站采用响应式设计，适配各种设备屏幕。

网站从最初的个人博客逐渐发展为一个综合的学习平台，包含交互式算法演示、项目展示和学习资源分享。

## ✨ 功能特点

### 🌐 主要页面
- **首页**：个人介绍、学习状态和B站空间展示
- **学习笔记**：CSAPP实验报告、数据结构学习心得和技术文档
- **日记**：个人日常记录，支持日历查看
- **交流页面**：友链展示，方便与朋友们互相交流
- **图片墙**：个人图片收藏和meme图集
- **产品展示**：开发的应用程序和学习工具展示
- **📝 文档编辑器**：可在线编辑Markdown文档的动态系统（支持Docker部署）

### 📚 技术文档与工具
- **图算法可视化**：Prim、Kruskal、拓扑排序、AOE关键路径算法的交互式演示
- **KMP算法可视化**：字符串匹配算法的详细演示
- **NeoChat**：局域网聊天程序介绍和使用指南
- **在线代码空间**：编程环境配置指南
- **CDN使用指南**：静态资源优化实践
- **Docker部署指南**：动态文档系统的容器化部署
- **不确定性理论**：相关学习笔记
- **树与二叉树算法**：数据结构可视化

### 🎨 设计特色
- **响应式布局**：完美适配桌面、平板和手机
- **统一导航**：通过JavaScript动态加载通用header
- **现代UI设计**：卡片式布局、悬停动画和过渡效果
- **深色主题**：以紫色为主色调的现代设计风格
- **交互式演示**：算法可视化支持单步执行和实时反馈

### 🎵 多媒体功能
- **背景音乐播放器**：网站氛围音乐
- **图片网格展示**：支持多种布局和比例显示
- **PDF在线预览**：学习笔记和文档的便捷查看

## 🚀 快速开始

### 方式一：Docker 容器化部署（推荐）

**适用于生产环境，包含完整的前后端功能**

```bash
# 1. 克隆项目
git clone https://github.com/FunctionHookTJU/fxHook.io.git
cd fxHook.io

# 2. 切换到包含后端的分支
git checkout test-SQLite

# 3. 一键部署
docker compose up -d --build

# 或使用快速部署脚本
bash quick-deploy.sh

# 访问网站
# 主站: http://localhost/
# 博客: http://localhost/blog-system/
# 文档: http://localhost/docs/
# 编辑器: http://localhost/editor/
```

详细部署文档请查看 [DOCKER_DEPLOYMENT.md](DOCKER_DEPLOYMENT.md)

### 方式二：本地开发

**适用于本地预览和开发**

#### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 支持ES6+ JavaScript特性
- Node.js 18+ (仅博客系统需要)

#### 使用方法
1. 克隆或下载本项目到本地
2. 确保所有文件保持原始目录结构
3. 使用浏览器直接打开`index.html`，或通过Live Server等工具启动本地服务器
4. 开始浏览和学习！
